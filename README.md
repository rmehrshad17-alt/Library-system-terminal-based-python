A simple **Python** library management system that allows you to add, view, search, edit, and remove books.  
Designed as an interactive console application using dictionaries to store data.

---

## Features

- Add books with **name, writer, release date, and count**.  
- Automatically generate a **unique ID** for each book.  
- Display all books in a neat and readable format.  
- Search books by **name** or **ID**.  
- Edit book information (name, writer, release date, count).  
- Remove books from the library.  
- Interactive text-based menu for easy use.

## Notes

Book ID is generated automatically and guaranteed unique.

Searching by name is case-sensitive.

Suitable for educational purposes or small library management.

Possible Enhancements

Save and load books from a JSON file or database.

Add categories or genres for books.

Create a GUI using Tkinter or PyQt.

Support multiple users with login functionality.
